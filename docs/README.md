## システム概要

本システムは以下の2つのサブシステムから構成されています。

### A1. プロジェクト管理システム（Web）
- プロジェクトの登録・更新・照会・検索機能
- プロジェクトのアップロード・ダウンロード機能
- Cron Jobによるプロジェクト一括登録・出力機能

### B1. 顧客管理システム（REST API）
- 顧客の登録・更新・照会・検索・削除機能
- Next.js API Routes形式での提供

## 技術スタック

| カテゴリ | 技術 |
|---------|------|
| フレームワーク | Next.js 14 (App Router) |
| 言語 | TypeScript |
| UIライブラリ | React 18 |
| スタイリング | Tailwind CSS |
| 状態管理 | Zustand / React Query |
| フォーム | React Hook Form + Zod |
| API | Next.js API Routes / Server Actions |
| ORM | Prisma |
| 認証 | NextAuth.js |
| テスト | Jest + React Testing Library + Playwright |
| データベース | PostgreSQL |

```

├── app/                    # App Router
│   ├── (auth)/            # 認証関連ページ
```
