# ソースコード自動生成設計書

本ドキュメントは、Spring Boot版サンプルプロジェクトのソースコードを静的解析して自動生成した設計書です。

> **生成方法**
>
> ソースコードの構造（パッケージ、クラス、メソッド、アノテーション、SQL）を解析し、
> 人手による仕様書なしで設計情報を抽出しています。

## 抽出可能な情報と限界

### 抽出できる情報

| 項目 | 抽出方法 |
|------|----------|
| クラス名・メソッド名 | Javaファイル解析 |
| URLマッピング | @RequestMapping等のアノテーション |
| フォームフィールド | Form/DTOクラスのフィールド定義 |
| バリデーションルール | @NotNull, @Size等のアノテーション |
| テーブル構造 | Entity/Modelクラス、DDL |
| バッチジョブ定義 | @Configuration, Spring Batch設定 |

### 抽出が困難な情報

| 項目 | 理由 |
|------|------|
| 業務要件・背景 | コードに記載されない |
| 画面レイアウト詳細 | HTMLテンプレートからの推測に限界 |
| 非機能要件 | 明示的なコードがない場合が多い |
| 運用手順 | ドキュメント外の情報 |
| テストシナリオ詳細 | テストコードからの推測に限界 |
