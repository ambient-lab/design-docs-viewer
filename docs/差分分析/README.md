# 差分分析ドキュメント

## 概要

このセクションでは、**ソースコードから自動生成された設計書**と**既存の設計書（ソースコード+仕様書から作成）**の差分を分析します。

## 目的

1. **ソースコードのみ**からどこまでの設計情報が抽出可能かを検証
2. **仕様書（要件定義、外部設計等）** がないと得られない情報を明確化
3. 設計書自動生成ツールの限界と活用方法の検討
